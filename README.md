# RNA-seq Analysis Pipeline

## Overview
This repository contains scripts for RNA-seq data processing and analysis, including:
- Quality control
- Alignment
- Gene expression quantification
- Differential expression analysis

## Requirements
- Python 3.9+
- R (for DESeq2)
- Tools: FastQC, STAR, HTSeq-count
